Instalando o Git

Passo a Passo

  1. Baixe o arquivo do Git no site: https://git-scm.com/downloads
  2. Configure o Git com sua Conta no Github utilizando os comandos a seguir no cmd ou Git.bash
    • git config --global user.name "SeuNome"
    • git config --global user.email "SeuEmail"
