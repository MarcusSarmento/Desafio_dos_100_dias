Vinculando o Git

Passo a Passo

  1. Crie um repositório no seu Github.
  2. Copie o link do repositório que deseja vincular ao Git.
  3. Vincule um repositório do Github no Git utilizando o comando a seguir no cmd ou Git.bash
    • git remote add origins https://github.com/URL_do_seu_repositorio
