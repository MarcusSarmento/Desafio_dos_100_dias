Introdução e instalação do React

Passo a Passo

  1. Baixe o Node.js. Verifique se foi baixado corretamente utilizando o comando"node -v" no VSCode.
  2. Insira o comando "npm create vite@lastet e siga o processo de criação de um projeto React.
  3. Após a instalação, digite os seguites comandos em ordem:
    • cd nome_do_projeto
    • npm install
    • npm run dev
  4. Tambêm e possível acessar e enviar comandos do Git.bash pelo terminal do VSCod.e
