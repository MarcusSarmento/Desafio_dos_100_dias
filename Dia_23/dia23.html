Commit & Push

Passo a Passo

  1. Após o Git reconhcer as mudanças, digite o código "git comit -m "descrição"" no terminal para comitar as descrições.
  2. Digite o código "git push -v origin master" para restaurar as ultimas mudanças caso precise.
