Iniciando Projeto em Django

  1. Beixe o Django utilizando o comando: pip install django
  2. Imsira os comandos:
    • django-admin startproject nome_do_projeto; para criar um novo projeto.
    • cd nome_do_projeto; para entrar na pasta do projeto.
    • python manage.py runserver
    • python manage.py startapp blog
  3. Adicione a pasta "blog" em settings.py na pasta do projeto.
  4. Adicione o caminho "path('', include('blog.urls'))," em urls.py
  5. Crie o arquivo "urls.py" na pasta blog
