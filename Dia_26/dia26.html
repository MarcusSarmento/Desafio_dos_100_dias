Git Fork

Passo a Passo

  • É possível copiar um repositório de terceiros sem precisar clona-lo.
    1. Selecione o repositório que deseja copiar.
    2. Clique no botão Fork na página inicial do repositório.
    3. Indique o local que deseja armazenar esse repositório.
    4. É possível editar o repositório de outra pessoa no seu computador, ideal para projetos em grupo.
